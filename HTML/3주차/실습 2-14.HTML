CITE 태그는 참고자료 등을 표시 할 때 사용한다.
CODE 태그는 컴퓨터 코드를 표시할 때 사용한다.
SMAP 태그는 프로그램 등의 샘플 코드를 표시할 때 사용한다.
VAR 태그는 변수명을 표시할 때 사용한다.
DFN 태그는 용어에 대한 정의를 표시 할 때 사용한다.
EM 태그는 강조하는 내용을 표시 할 때 사용한다.
STRONG 태그는 강조하는 특정부분을 표시 할 때 사용한다.
ABBR 태그는 축약형을 표시 할 때 사용된다.
ACRONTM 태그는 두 문자어를 표시 할 때 사용한다.
ADDRESS 태그는 문서와 관련된 연락 정보를 표시 할 때 사용한다.

KBD 태그는 키보드로 입력한 내용임을 표시 할 때 사용된다.
Q 태그는 인용한 단어 등을 표시할 때 사용된다.
